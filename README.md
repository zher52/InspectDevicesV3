# Inspect Devices V3 Chrome 扩展

## 项目简介
本扩展用于快速打开 Chrome 的设备检查页面（`chrome://inspect/#devices`），方便开发者调试连接的设备。

## 功能特性
- 点击扩展图标直接跳转至设备检查页面
- 包含16x16、48x48、128x128三种尺寸的图标（位于`icons/`目录）
- 采用MIT协议开放源代码

## 安装与使用
1. 在 Chrome 浏览器中打开扩展程序管理页面（`chrome://extensions/`）
2. 启用「开发者模式」（右上角开关）
3. 点击「加载已解压的扩展程序」
4. 选择本项目根目录
5. 扩展图标将出现在浏览器工具栏，点击即可使用
